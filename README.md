# REST API

Contains

- Get all students (GET `/students`)
- Get a student with an ID (GET `/students/:id`)
- Add a new student (POST `/student`)
- Update existing student information (PUT `/student`)
- Delete a student record (DELETE `/student/:id`)

# Setup

- Clone the repo
- `go run main.go`
